# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Recrify, please report it privately instead of opening a public issue.

Send reports to:

**recrify@gmail.com**

## What to Include

To help with investigation, please include as much relevant detail as possible:

- A clear description of the issue
- Steps to reproduce the behavior
- The affected area, endpoint, or file if known
- Potential impact
- Any proof-of-concept details that help validate the issue

## Disclosure

Please allow reasonable time for the issue to be reviewed and addressed before making any public disclosure.

## Scope

This policy applies to security issues affecting the code, integrations, and deployment-related behavior of this repository.
